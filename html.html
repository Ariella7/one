Html에 대해서 알아보자

Posted by Ariella's 일상 이야기

html, head, body

html – 문서의 시작과 끝을 묶는 태그.
head – 본문의 내용에 대한 부가정보 의미
body – 본문의 내용 의미

a 태그

anchor 약자, 닻 의미(어떤 페이지로 연결시키느냐)
* 반드시 속성 값 넘겨주어야 함

li 태그

list 약자, = 목차 생성

ol 태그

ordered list 약자, ol 또는 ul은 li의 부모태그로 감싸져 있어야 한다.

p 태그

pragraph 약자, = 문단 지정(문단 나눌 때 사용)

br 태그

line breaker 약자, = 줄바꿈 태그

strong 태그

진하게 표시하고 싶은 부분

u 태그

nderline로 밑줄그을 때 사용한다.

img 태그

img src="원하는 이미지 주소 width="100"
width는 사진이 너무 클때 줄이기 위해 사용
