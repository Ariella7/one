주석

!-- ~~~내용~~~ --를 사용하게 되면 웹브라우저가 내용을 인식하지 못한다.

;(=세미콜론)

시작하고 끝나는 지점을 구분하기 위해 사용한다.

style

css틀의 기본

글씨 크기

예시) font-size: 250%; / font-size:60px;(px=픽셀)

가운데/왼쪽/오른쪽 정렬

예시) text-align:center;/ text-align:left;/ text-align:right;

테두리

예시) border-width:5px; - 두께
border-style: solid; - solid 단선/ 실선/ 점선

상자(컨텐츠) 안의 여백

예시) padding:20px;

상자와 상자의 여백

예시) margin:0;

block(tag,컨텐츠) 크기 변경(width=폭, height:높이)

예시) width:100px;

태그나 클래스가 아니라 디자인만을 목적으로 사용하는 언어

1. div - 화면전체 2. span - 자신크기만큼 사용

*선택자

* class가 여러개인 것 보다 ID로 넣어주는게 더 좋다.
** 동일한 level 일 때는 나중에 쓴 선택자가 우선 = 마지막 선택자가 우선순위가 된다.(제일 영향력이 세다=우선적용된다.)
*** 다른 level 일 때는 id > class > tag 순
**** id는 해당 문서에서 한번밖에 사용할 수 없음, id의 값도 중복되서는 안된다.
